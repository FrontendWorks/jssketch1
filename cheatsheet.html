Keyboard Shortcut Cheat Sheet for Sketch

Insert

  • Artboard: A
  • Rectangle: R
  • Rounded Rectangle: U
  • Oval: O
  • Line: L
  • Vector: V
  • Pencil: P
  • Text: T
  • Slice: S

Editing Shapes

  • Keep Current Selection: command+option
  • Use as Mask: command+control+M
  • Resize Object: command+arrows
  • Resize Object (by 10px): command+shift+arrows
  • Union: command+option+U
  • Subtract: command+option+S
  • Intersect: command+option+I
  • Difference: command+option+X
  • Change Vector Point Style: 1, 2, 3 or 4

Window

  • Toggle Between Documents: command+~
  • Toggle Layer List: command+option+1
  • Toggle Inspector: command+option+2
  • Layer List & Inspector: command+option+3
  • Toggle Toolbar: command+T
  • Presentation Mode: command+.
  • Fullscreen Mode: command+control+F

Type

  • Bold: command+B
  • Italic: command+I
  • Underline: command+U
  • Inc. Font Size: command+option+(+)
  • Dec. Font Size: command+option+(-)
  • Inc. Character Spacing: control+option+L
  • Dec. Character Spacing: control+option+T
  • Change Font: command+T
  • Align Left: command+shift+{
  • Align Center: command+shift+|
  • Align Right: command+shift+}
  • Special Characters: command+control+space
  • Convert Text to Outlines: command+shift+O

Canvas

  • Zoom In: command+(+)
  • Zoom Out: command+(-)
  • Actual Size: command+0
  • Center Canvas: command+1
  • Zoom Selection: command+2
  • Center Selection: command+3
  • Temp. Zoom to Actual Size: ยง or shift+~
  • Focus on First Input Field: option+tab
  • Toggle Rulers: control+R
  • Toggle Grid: control+G
  • Toggle Layout Guides: control+L
  • Toggle Pixels: control+P
  • Toggle Selection Handles: control+H
  • Toggle Pixel Grid: control+X
  • Move Canvas: space+drag

Editing Layers

  • Show Smart Guides: option
  • Click-Through to Group: command+click
  • Smart Guides + Click-Through: command+option
  • Duplicate: command+D or option+drag
  • Copy Style: command+option+C
  • Paste Style: command+option+V
  • Color Picker: control+C
  • Transform: command+T
  • Rotate: command+shift+R
  • Toggle Fill: F
  • Toggle Border: B

Arranging Objects

  • Bring Forward: command+option+up
  • Send Backward: command+option+down
  • Bring to Front: command+option+control+up
  • Send to Back: command+option+control+down
  • Hide: command+shift+H
  • Lock: command+shift+L
  • Rename: command+R
  • Group Layers: command+G
  • Ungroup Layers: command+shift+G
  • Select Layer Above: shift+tab
  • Select Layer Below: tab
  • Select Parent Artboard: esc
  • Find Layer by Name: command+F
  • Select Page Above: fn+up
  • Select Page Below: fn+down
